# 🎬 TikTok Script 10 คลิป — AlphaScout

## 🎯 หลักการสร้างคลิปไวรัล (อ่านก่อน)

### โครงสร้าง 3 วินาทีแรก (Hook) — สำคัญที่สุด
- คลิปแพ้-ชนะตัดสินใน 3 วินาทีแรก
- **ห้ามแนะนำตัว**, **ห้ามอธิบายยาว** → เข้าเรื่องเลย
- ใช้ Pattern Interrupt: คำถามแรง / ตัวเลขใหญ่ / สิ่งที่ขัดความเชื่อ

### โครงสร้าง 1 คลิป (15-60 วินาที)
```
[0-3s]   HOOK — ดักความสนใจ
[3-15s]  PROBLEM — ขยายแผล
[15-45s] SOLUTION — โชว์ AlphaScout
[45-60s] CTA — "กดลิงก์ใน Bio"
```

### Format ที่ Algorithm ชอบ (2026)
- **9:16 vertical** (1080x1920)
- **คำบรรยายเด่น** ที่ขอบบน ใช้คำเดียว/สั้นๆ
- **กล้องขยับ** ทุก 1-2 วินาที (ใส่ effect, zoom, cut)
- **เสียงพูดดัง คมชัด** ไม่ใช้เพลงเสียงดังกว่า
- **เสียงเทรนด์** ใช้เสียงที่กำลังฮิต (เปิด TikTok Creator Center ดู)

## 🔥 Tips จาก Top Tier Marketer

1. **โพสต์ 2-3 คลิป/วัน** (เช้า/บ่าย/เย็น) ใน 30 วันแรก
2. **อย่าลบคลิปเก่า** TikTok อาจดันคลิปเก่าให้ไวรัล (delayed virality)
3. **Comment ตอบทุกคำถาม** ใน 30 นาทีแรก — Algorithm ชอบ
4. **Reply Comment ด้วยวิดีโอ** (Feature TikTok) → คลิปใหม่ฟรี
5. **อ้างอิงข่าวจริง** ทุกครั้ง (เช่น "NVDA วันนี้") → คนสนใจ
6. **Voice over ภาษาไทยชัด** ไม่ใช้สำเนียงต่างประเทศ
7. **ใช้ Trend Sound** เพิ่ม Reach 3-5 เท่า
8. **อย่าลงลิงก์ในคำบรรยาย** — บอก "Bio" เท่านั้น
